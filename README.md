##  Project Overview

A Digital Public Good (DPG) is a digital system designed to solve societal problems and improve people's lives, often aligned with the United Nations Sustainable Development Goals (SDGs).

This project presents the **design** (not development) of **Limko Wi Skul**, a student academic management and results system built to address inefficiencies in record-keeping, attendance tracking, and communication within Limkokwing University, Sierra Leone.

**SDG Focus:** Goal 4 – Quality Education

## System Summary

**Limko Wi Skul** is a mobile academic system serving four user roles:

- **Student** – views results, attendance, timetable, notices, and notifications
- **Lecturer** – takes attendance and submits results
- **Registry Officer** – verifies and publishes official results
- **Dean** – monitors departmental academic performance

### Key Features
- Role-based secure login
- Real-time results and GPA tracking
- Digital timetable
- Centralised announcements and notifications
- Attendance management for lecturers
- Departmental performance reporting for the Dean

##  SDG 4 Alignment

Limko Wi Skul supports **SDG 4: Quality Education** by:
- Reducing delays and errors in academic record-keeping
- Improving transparency between students, lecturers, and administration
- Centralising communication of important academic information
- Enabling data-driven academic decision-making
